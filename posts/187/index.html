CDNJS - Web 上最快的 Javascript 资源库

Jul 25th, 2015 10:59 am

CDNJS 是一个互联网上的 JavaScript 资料库，类似 Google CDN 和微软CDN服务，但是速度比这二者更加快。CDNJS 上提供了众多 JavaScript 库，你可以直接在网页上引用这些 JS 文件，实现用户浏览网站的最佳速度体验。

CDNJS 上提供的完整的 JavaScript 库列表请看 http://cdnjs.com/

这些库是开源的，你可以提交自己的 JavaScript 库到 CDNJS，只需要提交到 CDNJS 在 Github 上的项目即可

https://github.com/cdnjs/cdnjs/
