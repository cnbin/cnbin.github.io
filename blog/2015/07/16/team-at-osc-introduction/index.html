Team@OSC Introduction

Jul 16th, 2015 10:21 am

Git 集成

可以与 GitHub 和 Git@OSC 项目集成，项目提交、Issues 等更新信息一目了然。

任务&BUG管理

任务&BUG管理是team的核心，可以是一个简单的任务，BUG，或者是一个新功能，新需求。

项目协作文档

多人在线协作文档，简单的文件结构和Markdown来创建自定义格式，另外还有文档主题随时切换，帮助你轻松管理以及快速生成漂亮的项目文档

团队讨论

高效率的在线讨论功能，让每个人都能随时参与讨论，迅速找到最佳方案。

资源分享

把文件拖到分享区的网页上即可完成共享，更可以对文件进行分类整理。

便签

随时随地记录你的灵感与想法！不同的颜色记录你不同的心情，更多体验等你来玩！

项目周报

简单直观的项目简报，让你对项目的进展了如指掌。帮助你轻松地记录、报告工作，减少相关会议的邮件。
