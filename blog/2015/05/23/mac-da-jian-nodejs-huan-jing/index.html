Mac 安装 SublimeText

May 23rd, 2015 4:43 pm

安装nodejs

在 Mac下，如果你喜欢用 homebrew，那么只用一行就可以装好: 

brew install node 

否则，只能考虑手工安装了，步骤如下： 

1.安装 Xcode 

2.安装 git 

3.运行下面的命令行编译 node.js 

4.复制代码代码如下:

git clone git://github.com/ry/node.git 
cd node 
./configure 
make 
sudo make install 

安装插件

1.下载Sublime Text：http://www.sublimetext.com。终身免费。界面炫酷，简单小巧。

2.Sublime 默认没有 Nodejs 语法高亮与自动补全，需要下载 Nodejs 插件：https://github.com/tanepiper/SublimeText-Nodejs。
