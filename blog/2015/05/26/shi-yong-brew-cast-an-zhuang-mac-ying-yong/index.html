使用 Brew Cast 安装 Mac 应用

May 26th, 2015 11:38 am

小引

brew cask是一个用命令行管理 Mac 下应用的工具，它是基于homebrew的一个增强工具。

homebrew 可以管理 Mac 下的命令行工具，例如 imagemagick, nodejs，如下所示：

brew install imagemagick
brew install node

而使用上 brew cask 之后，你还可以用它来管理 Mac 下的 Gui 程序，例如 qq, chrome, evernote 等，如下所示：

brew cask install qq
brew cask install google-chrome
brew cask install evernote

安装

安装homebrew

只需执行命令：

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

之后执行 brew doctor 命令可以看看 homebrew 的环境是否正常。通常第一次安装完 brew 之后，还需要安装苹果的Command Line Tools。

安装cask

用如下命令来安装 cask:

brew tap phinze/cask
brew install brew-cask
