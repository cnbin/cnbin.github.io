高效的 iOS 和 OS 开发工具

May 30th, 2015 8:49 am

1.Anvil（免费）–Anvil 是个漂亮的菜单栏 app，可以用来管理本地网站。

官网

2.Go2Shell –非常实用的小工具，可在任何文目录中打开终端窗口进行操作。

App Store下载

3.Hex Color Picker–可帮你方便地抓取 NSColor 或 UIColor 值

官网

Hex Color Picker 并不是一个独立的应用程序，而是嵌入在 Mac OS X 原本的色彩检测面板中的小工具，只要将 Hex Color Picker 程式下载回來、解压缩之后，把「HexColorPicker.colorPicker」档案复制到硬盘的「资源库」→「ColorPickers」资料夹中即可使用，下次再开启 Mac 系統內建的色彩检测面板时，就可看到多了一个 Hex Color Picker 分页。

4.iTerm2–Mac上Terminal app 的替代选择，在原先的基础上增添了很多神奇的功能，包括搜索、自动完成，无数配置选项以及 TotalTerminal 等等，也是 GitHub 上的开源项目。

官网

5.SourceTree– Windows 和 Mac 下免费的 Git 和 Hg 客户端。

官网

6.GitHub for Mac–在 Mac上更简单地使用 GitHub 。

官网
