Coding 轻松搭建 Z-Blog

Jun 19th, 2015 6:17 pm

下载 Z-Blog 程序安装包

方法一：从 github 上下载

Z-Blog github 下载

方法二：从官网下载

Z-Blog 官网

下载解压后，修改文件夹权限。

上传程序代码

进入 zblogphp-master 文件夹，然后执行命令：

$ git init
$ git add .
$ git commit -m "first commit"
$ git remote add origin https://git.coding.net/cnbin/zblog.git
$ git push origin master

这里 git remote 之前要现在 Coding 中创建一个项目。

部署演示平台

一.添加服务

添加好服务好，记下数据库信息，待会安装 Z-Blog 要用到

二.部署控制台

点击控制台，自定义域名，应用内存，一般部署版本是 master，然后一键部署，当显示绿灯时，既可以马上访问。

进入 Z-Blog

进入 Z-Blog 后台，既可以根据自己的需求自定义博客系统
