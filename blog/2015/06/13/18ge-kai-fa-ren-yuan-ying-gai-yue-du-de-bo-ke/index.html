18个 Web 开发人员应该阅读的外国博客

Jun 13th, 2015 8:31 am

1.Six Revisions

我们不再把 Six Revisions 成为一个普通的 WEB 开发网站，而是开发人员的新闻网站。Six Revisions 每2-5天会发布一篇对开发者和设计者实际有用的文章。

2.WebResourcesDepot

WebResourcesDepot 是一个很大的开发博客，每天发布新的文章。开发者在这里肯定会找到非常有用的内容。

3.Smashing Magazine

Smashing Magazine 是另一个不容忽视的著名的新闻网站。每个一天发布一篇新文章，为你提供大量与 HTML、CSS、JavaScript 有关的资源。

4.David Walsh

这个网站会提供大量 CSS、jQuery、JavaScript、PHP、MySQL 等有关的文章。

5.Stoyan’s phpied.com

一个非常值得关注的个人博客，作者是 Facebook 的工程师，很多有名的书的作者。是一个发现新文章、工具、框架的好地方。

6.OnextraPixel

我觉得与其说 OnextraPixel 是一个开发博客，不如说它是一个设计博客。然而，它也提供大量开发领域有用的文章。无论是关于HTML5的有用的文章，还是关于 jQuery 的一些小建议。你肯定会得到一些启发性的内容。

7.Alex Sexton

Alex Sexton 的个人博客用一种简单易懂的方式介绍了各种各样的 JavaScript 方面的知识。你也可以在他的博客上发现一些新的工具和趋势

8.Scotch.io

Scotch.io 发布一下 AngularJS、node、JavaScript、Bootstrap 以及其他方面的一些文章、教程和视频。这绝对是一个你寻找新的灵感和开发资源的好地方。

9.Net Tuts

Net Tuts是一个高品质的设计博客，发布技术，技巧方面的文章和教程，内容涉及 HTML、CSS、 Javascript、CMS、ASP.net、PHP、Ruby 以及数据库等。

10.We Function

wefunction 是一个非常好的设计博客，发布非常有用的 Web 设计和开发方面的资源和教程，还有 Wordpress 主题等。

11.Fuel Your Creativity

Fuel yourc reativity 顾名思义是重点关注创意的一个设计博客，内容主要有创意工具、技术发展趋势以及免费资源等。

12.Smashing Apps

Smashing apps主要向 Web 设计和开发者分享非常有用的免费在线工具，另外还有很多其它的免费教程和资源。

13.Think Vitamin

Think vitamin 创建于2006年，内容涉及 JavaScript、HTML、CSS和Ruby on Rails 等方面。

14.Instant Shift

Instant shift 是 Web 开发人员最喜欢去的网站，内容涉及网页设计、图标、免费工具、壁纸、Wordpress 等等。

15.Web designer

Web designer 的作者是 Nick La，拥有丰富的设计经验，其博客内容涉及设计思想、教程和现代设计趋势等方面。

16.Designm.ag

DesignM.ag 成功的为 Web 设计和开发人员建立了一个基于社区的平，和其它博客一样有很多很多吸引力的内容。

17.Noupe

Noupe 由 Noura Yehia 创建于2007年，内容涉及 CSS、Ajax、JavaScript、Photoshop、Wordpress 以及网页设计等。

18.Abduzeedo

Abuzeedo 由F abio sasso 创建于 2006 年，是最流行的设计博客之一，内容主要是关于设计教程、壁纸以及访谈。
