Mac 安装 Gulp(原创)

Jun 10th, 2015 8:53 am

gulp 是一个 node.js 模块，也是前端自动化任务构建工具。

使用 gulp，可以做以下工作：文件拷贝、文件合并、文本替换、图片压缩、CSS压缩、js压缩等等，当然这些工作主要是依靠 gulp 的相应插件来完成的。

安装

官网

sudo npm install -g gulp

查看 Gulp 版本

gulp -v

使用 gulp 搭建 web 服务器

（1）初始化安装

npm install --save-dev gulp gulp-connect

完成后会把相应的 npm 模块（gulp、gulp-connect）安装在项目目录下。

（2）创建 gulpfile.js 配置文件

var gulp = require('gulp');
var connect = require('gulp-connect');

gulp.task('webserver', function () {
    connect.server();
});

gulp.task('default', ['webserver']);

（3）创建一个默认首页 index.html

<h1>hello gulp!</h1>
