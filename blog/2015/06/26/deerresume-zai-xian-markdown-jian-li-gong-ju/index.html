DeerResume : 在线 MarkDown 简历工具

Jun 26th, 2015 2:42 pm

DeerResume 是一款支持实时预览、阅读密码、一键 PDF 生成的在线 MarkDown 简历工具。

本文包括两个部分，特点介绍及安装指南。

特点介绍

拥有以下突出特点：

  • 简单低调又不失雅致的界面
  • 可自行搭建，细节可控
  • 猴子都会安装
  • 在线编辑，实时预览
  • 阅读密码，只有你愿意的人才能看到你的简历

特点一：简单低调又不失雅致的界面

Demo → http://deercv.sinaapp.com/ 阅读密码 4444

特点二：可自行搭建，细节可控

虽然你在几乎所有招聘站都有简历，但没有哪个你能自己控制，哪怕是改改样式，调调细节。如果你使用DeerResume 来搭建你的在线简历，那么你可以控制它的每一个细节。这里是它在 GitHub 的 Repo → https://github.com/geekcompany/DeerResume

特点三：猴子都会安装

不需要配置 PHP、MySQL，更不需要挂上 PDF 转换模块。DeerResume 只需要将项目文件放到可通过浏览器访问的静态空间即可完美运行，是的，GitHub 就可以。

为了将易用性做到最好，我们使用了云服务来做数据可控性和转换 PDF，如果您对此不放心，可以在下载完 PDF 后，手动清空简历数据。

特点四：在线编辑，实时预览

使用指南

电梯

如果你不想手工部署，可以使用 DeerResume Online，10分钟轻松写完简历 → http://cvbox.sinaapp.com

下载

首先，到 GitHub 下载 DeerResume 源代码，→ https://github.com/geekcompany/DeerResume/archive/master.zip

上传

然后，传到你希望在线浏览的 Web 目录下，只要支持静态文件就行。注意简历的数据和访问路径有依赖关系，通过路径A提交的简历，只能在路径A下查看。如果你要在本地试用，请放到类似 http://localhost/yourname 的访问路径下。

进入管理界面，设置密码

上传完成后，DeerResume 就可用了，访问它会显示默认的简历模板。这时候要赶紧给它设置管理密码。点击右上方的小齿轮图标，可以进入管理界面。在实时预览的编辑器上方，可以设置在线简历的标题、子标题、管理密码和阅读密码。

管理密码只能设置一次，不可修改，请谨慎设置。阅读密码为空的话，所有人都可以看到你的简历。所有密码长度都必须大于3个字符。

在线浏览和 PDF

当你修改完成后，可以点击「 保存 」按钮存档，点右上的「返回首页」查看实际效果。点击首页右上方的图书按钮可以下载 PDF 版本。

DeerResume 还对手机做了适配，保证简历在 Android 和 iPhone 上有不错的阅读体验。
