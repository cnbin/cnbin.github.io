Linux - Cd 命令

Jun 26th, 2015 2:47 pm

指令：cd 切换当前目录至目标目录

语法：# cd [目录名]

使用范例 1、进入到 /home/apple/Public 目录

# cd /home/apple/Public

查看当前所在目录

# pwd

2、进入到当年目录中的 Public 目录

# cd Public

其实上面两个例子涉及到相对路径以及绝对路径的问题，详细请参阅相对路径与绝对路径

cd 命令使用小技巧：

# cd      进入用户主目录
# cd ~    进入用户主目录
# cd -    返回进入此目录之前所在的目录(即上层目录)
# cd ..    返回上级目录，若当前目录为/目录，则执行完后还在/目录
# cd !$    把上个命令的参数作为cd参数使用
