Jekyll 静态博客生成工具

Jun 21st, 2015 9:14 am

Jekyll 是 一款非常出色的静态博客网站生成工具，它是用 Ruby 开发而成的。和 Wordpress 相比，Jekyll 更为轻量，它没有动态内容，也没有数据库连接， 只是用来快速生成静态页面。目前越来越多的博主都选择在 Github 中使用 Jekyll ，并且绑定一个域名就可以实现一个免费的博客空间。

jekyll 有以下特点

  • 完全免费开源，基于 Ruby，效率很不错。
  • 默认支持 markdown 语法，相比 HTML 更为简洁。
  • 无需数据库，完全本地文件同步。
  • Github 支持 Jekyll，可以免费获取个人博客空间。

安装jekyll

sudo gem install jekyll

检验是否安装成功

jekyll –version

安装 Rdiscount，这个用来解析 Markdown 标记的包，使用如下命令：

gem install rdiscount
