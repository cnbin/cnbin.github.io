致青春-2

Jun 1st, 2015 2:52 pm

5.25到5.30，总结下这周做了什么？

看了很多内容，废话一句。这周过得很充实，今天是31号，给自己定了一个新的目标。就是每天无论如何要坚持写5篇技术博文。为什么呢？

做软件是一件长期的事，特别是公司目前给了我一个机会去积累，但是由于对一个只学过 C 语言，玩过 PIC 单片机，会点 Java 的小伙子来说，代码量连1万句都还没突破。很幸运，这个月下来，自己打代码的速度有提高了，键盘电池也被我换了。如果每天坚持写5篇，到今年结束，就会有1000篇博文，然后再坚持3年，也就是到2017年，可以达到5000篇。为什么要坚持呢？因为技术更新很快，有些内容要做好笔记，一来自己可以当手册用，以后不用再走回头路，二来可以告诉别人我会什么了，在做什么，我对技术的热爱程度，三来知识就是用来分享和传承的，我一直想do something difference，以前初中是用 QQ空间写心情，写文章，后来高中就没劲不玩了。现在工作了，找到这个方法，也就当一个目标，每天最少学习5个新的知识点，并且及时做好总结，自己走出一条属于自己的路，欣赏沿途的景色，在知识的海洋里，我只取一瓢。希望多找到一些技术上的爱好者，共同进步。

这一周学了 Nodejs，Mongodb，Ruby on Rails，Django，Angular，这些 web 前端新概念。也用这些 web 框架试着搭建个博客，走了一遍，不玩虚的。毛主席说过，没实践过的没有发言权。github 账号也上传了 Rails_blog 和 Django_blog。目前由于 Nodejs Express 升级到 4.0，看了一本叫 Nodejs 开发指南的书，里面讲的是 EXpress2.0 的内容，很无奈，书有点过时了，最近刚在网上找到一个博主用 Express4.0 的，接下来要研究下。

另外，还学了 yeoman，grunt，bower 搭建了一个 Todo 应用，对自动化开发工具有了一个认识。虽然我的最终目的是做 iOS 项目，但是目前连一点 web 前端的最新技术和常识都不具备的话，能做好高大上的 iOS 开发吗？呵呵，做梦吧你。iOS 也是离不开 web 的。

接着，对 Markdown语法掌握比较熟练了，安装了 Sublime Text2编辑器，真是个神器，现在才知道这玩意，感觉很落后啊，还有 Emmet 插件，幸好自己发现，想起之前自己玩 Android 的时候，xml 布局写的代码真是打得蛋疼啊。越打越没劲，问题怎么解决？总算是找到一些 web 开发必备工具，不提高效率，注定与世界脱轨。据说现在都出来 Sublime Text3 测试版了，过段时间下载下来看看。

MAC 还安装了 skitch 截图软件，利用印象笔记记录要事。微博账号也重新启动，把一些好文章分享出去。简书也注册了账号，虽然目前还没有发表过文章。

论坛方面，加了 segmentfault，web前端。发现论坛里面水很深，自己很多方面都还需要学习。知识看来都是想通的。我学了 Nodejs 后，看起来没怎么用，但是后来学习其他的知识时，安装的前提就是要有 nodejs 环境。

现在的我，单身，技术男一个，正在学习 javascript，说这个感觉都笑死人了，是啊，长这么大，才来玩 javascript + html + css。但是没办法，也许是找到兴趣，但是实际点的，这是软件开发的基本要求，这个是必须要会的。我却只顾玩 C#，object-c这些了。

明天就是6.1儿童节了，跟我无关，我要做真正男子汉，继续努力，fighting！时间过得很快，眨眼就快半年了，2015，我在路上，前行，不要止步。
