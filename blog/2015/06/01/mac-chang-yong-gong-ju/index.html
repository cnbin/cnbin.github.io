Mac 常用工具

Jun 1st, 2015 11:09 am

软件管理

1.Homebrew

例子一：安装 go

brew install go

例子二： 安装 mongodb

brew install mongodb

例子三：安装 postgresql

brew install postgres

2.Cakebrew

如果您不习惯命令行，就使用 GUI 的 Cakebrew 。 Cakebrew 就是 The Mac App for Homebrew

开发者工具

1.pip pyton 的包管理工具，就像 ruby 的 gem

安装方法：sudo easy_install pip

2.n

node.js 版本管理工具

3.rvm

ruby 的版本管理工具

4.Dash

查看文档，管理代码段

5.CheatSheet

长按 command ，将能查看当前程序的快捷键

6.ShiftIt

管理窗口大小和位置，多屏幕/写代码/做设计/

7.RescueTime

可以监控你看了哪些网页，打开了哪些程序，用了多长时间。

8.Zoommy

一个寻找免费图库的应用 - all free stock photos in one place

Mac OS X系统：

1.软件开发以及数学工具：

Eclipse：跨平台跨语言集成开发环境，搞啥都行，开源免费，本人基本工具。
Xcode：苹果官方出品，没啥好说的。
TextWrangler: 据说是Mac OS 下最好的免费文本编辑器，必备。
HexFriend：十六进制编辑器，免费。
CodeCollect Pro: 把你有用的代码片段搜集起来，免费。
PSPP：嗯，就是一个类似SPSS的软件，你要不知道SPSS就不用管它，开源免费。
Sage：一大堆强大的开源数学软件的强力组合，搞偏数学的理论研究必备，开源免费。
Octave：就是matlab的开源替代品，强力兼容Matlab语法，科研必备，不知道Matlab的请无视，开源免费。
R：开源的统计分析处理语言，搞统计的必备，开源免费。
Gate：自然语言处理NLP强力工具，其他人员请无视，开源免费。
SciLab: 一个类似于Matlab的科学计算工具，但与Matlab语法并不一致，寻找Matlab替代品的请使用Octave，开源免费。
Waka：数据挖掘工具，其它人员请无视，开源免费。
Processing: 数据可视化工具，开源免费。
Visual Paradigm for UML: 完整的UML建模工具，Community版功能仍然十分强大，最新UML标准，免费。
Pencil：软件UI原型设计工具，免费。

2.网络及社交工具

Google Chrome：最爱浏览器，免费。
QQ：国内没它还真不行，免费。
Thunder：迅雷Mac版，一般般，就是支持BT和电驴，速度没有Win下好，并且不支持会员。免费。
Fake Thunder: 用它来搞迅雷的离线下载，免费。
阿里旺旺：淘宝控们必备。免费。
Skype：国外用户较多。免费。
DropBox：云存储，个人感觉不错，大量软件内嵌支持。免费。
Adium：支持众多协议的集成聊天工具，但是貌似QQ传文件不能？和国外朋友聊天方便。免费。
TunelBlick: OpenVPN的Mac客户端，免费。

3.办公软件

LibreOffice：办公软件套装，对MS Office系列兼容不错。开源免费。
Skim：PDF阅读神器，强大的批注功能。免费。
Mendeley Desktop：强大的参考文献管理，可导入EndNote、Papers、Zotero的数据，更爽的是，支持将PDF文件拖入，自动提取标题等信息并从Google  Scholar等获取基本信息！写论文必备！免费。
FreeMind：思维导图工具。免费。
MacTex：Tex排版编辑套件。不知道Tex的请无视。开源免费。
iCHM：CHM格式阅读器。免费。
Ehon: 电子书籍管理，把你的电子书分门别类放在书架上，支持查找。支持epub，PDF等多种格式。免费。
EverNote: 就是印象笔记。云笔记。免费。

4.影音图像

Picasa：Google旗下看图软件，不错的人脸识别，可以自动识别哪些人出现在哪些照片上。免费。
VLC：全能影音播放器。免费。
MPlayerX：全能影音播放器。开源免费。
Plex：专业影音播放，播放效果极佳，支持格式非常全面，但其操作方式（键盘，全屏播放）适合外接大屏幕当家庭影院。免费。
Audacity：相当不错的音频编辑工具。开源免费。
XLD：音频转码工具。免费。
LessDJ：网络音乐收音机。我喜欢里面的法语歌。免费。
豆瓣FM：国内不错的网络音乐收音机。有一定的学习功能。免费。
Jahshaka: 视频编辑。开源免费。
BRL-CAD：美国海军出品，专业CAD工具。开源免费。
FreeCAD：界面更适合新手的CAD工具，但不如BRL-CAD强大。免费。
Blender：相当不错的3D建模渲染工具。开源免费。
VUE 10: 专业三维景观建模工具。有免费版。
GIMP：相当不错的平面图像修改工具。PhotoShop的开源替代品。曾用在《泰坦尼克》的制作中。开源免费。
Inscape：矢量绘图工具，AI等的开源替代品。开源免费。

5.系统工具

CoRD：远程登录Windows的必备工具，免费。
VirtualBox：SUN出品（Oracle算什么鸟）的虚拟机软件，性能不错，不想来回切换操作系统的可以装下。免费。
AppCleaner：软件卸载辅助工具，帮你清除软件放在其它地方的文件。免费。
The Unachiver: 解压缩软件。免费。
WaterProof：让你可以充分发挥系统内置防火墙的威力。非专业人士勿动。免费。
Avast：很不错防病毒软件。别以为Mac很安全。既有直接感染Mac的，也有利用Mac传播的。免费。非流氓。可靠。
GPGTools：PGP加密工具。开源免费。
TrueCrypt: 无比强悍的磁盘底层加密工具，可以创建虚拟加密盘（就是把一个文件作为加密盘），也可以对整个分区或硬盘进行加密（不影响启动）。加密强度极高（传说FBI都无能无力）。需要在计算机上保存机密文件的朋友必备。开源免费。

其他工具

1.SnappyApp

一个快速方便的截屏，分享工具。特点是将每次的截屏图片钉在桌面最上面。截屏的快捷方式是 Cmd + Shift + 2。

2.爱壁纸

专业壁纸应用，帮你更换桌面壁纸，海量。已支持 Retina 屏。

3.Leaf Mac版

Leaf Mac 版是 OS X 上一款 RSS 客户端程序，也是一款轻型阅读软件，界面很漂亮，UI 和交互都挺新颖，对于轻型阅读，新颖的交互形式很美观很大方。

4.Popclip

PopClip 将 iOS 上的一些特色带回至 Mac 上，操作十分的简单，只要反白文字之后就会跳出一些常用的功能，例如：复制、粘贴。

[插件集]http://pilotmoon.com/popclip/extensions/

5.Sip Color：小巧轻便的iOS取色工具
